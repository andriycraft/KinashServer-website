A simple node.js http server for local testing or building simple websites



What does theese options mean?


⚬ host - Means what ip the server will listen for

⚬ port - Means what port the server will listen for

⚬ authentication_url - Url for the authentication page. Set to null to disable

⚬ authentication_username - Username for the authentication page. Do not change if authentication is disabled

⚬ authentication_password - Password for the authentication page. Do not change if authentication is disabled

⚬ authentication_realm - Realm for the authentication page (shown in response headers). Do not change if authentication is disabled

⚬ authentication_url - Url for the authentication page. Do not change if authentication is disabled

⚬ authentication_file - File for the authentication page. Must start with ./public_html/[FILE_NAME]. Do not change if authentication is disabled

⚬ max_url_length - This is maximum limit for request url lenght

⚬ error[ERROR_CODE]page - This is error page for [ERROR_CODE]

⚬ disallowcrawlers - Disallow almost all crawlers from accessing your website.

⚬ enablebasicsecuritychecks - Enables protection from XSS, and some exploits. !!! We do not recomment to disable !!!
